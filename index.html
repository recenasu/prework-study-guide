HTML

  • Hypertext Transfer Markup Language (HTML) is a markup language used to create the structure of a webpage.
  • The head element contains information about the web page.
  • The body element repesents the visible content shown to the user.
  • The metadata contains data about the webpage that the browser uses to serve the page to users. Metadata is typically not seen by users.
  • The title element is a required element inside the head element that defines the title for the webpage.
  • The style element defines the CSS styles associated with the webpage.

Git

  • A free, open-source distributed version control system.
  • Enables multiple developers to work together on non-linear development.
  • git commands are used for:
    • initializing and cloning repos
    • working with snapshots and the git staging area
    • isolating work in branches and merging branches into the main branch
  • git commands always begin with git
  • Some git commands:
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
